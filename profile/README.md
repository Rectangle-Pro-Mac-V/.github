## Overview

Rectangle Pro Advanced is a professional window management application that organizes application windows with keyboard shortcuts, snap areas, custom layouts, and drag-to-edge snapping that brings comprehensive window tiling to macOS. It extends the basic window management that macOS provides with the precise positioning, saved layouts, and multi-display awareness that productivity-focused users need to maintain organized, efficient workspaces across multiple monitors.

Keyboard shortcuts snap windows to halves, thirds, quarters, and custom positions with configurable key combinations that make window positioning instant and precise. The shortcut system covers every common window arrangement pattern and supports cycling through sizes with repeated shortcut activation — pressing the left-half shortcut multiple times cycles through one-half, one-third, and two-thirds widths.

Snap areas activate when windows are dragged to screen edges and corners, displaying visual zones that show where the window will be placed when released. Custom snap areas can be defined beyond the default edges and corners, creating personalized snapping behavior that matches specific workflow requirements. The visual feedback makes precise placement intuitive.

Saved layouts capture complete window arrangements — the position and size of every window on every display — and restore them with a single shortcut or menu selection. Layouts handle the common workflow of switching between different workspace configurations for different tasks — a coding layout with terminal and editor side by side, a design layout with browser and design tool, a communication layout with email and messaging.

For users who manage multiple windows across multiple displays and want precise, efficient window organization without manual dragging and resizing, Rectangle Pro Advanced is the window manager that provides the keyboard shortcuts, snap zones, and saved layouts that productive multi-window workflows demand.

## Performance

Rectangle Pro balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
